The project
What is the purpose of this project ?
This project stands for a tool to help scientists to monitor mosquitoes propagation.
To achieve this goal, our tool is retrieving various data:
  • Mosquito species guessed from its picture, using Machine Learning
  • Geolocation, asked from the user and used to retrieve other data as:
  • The mosquito population density
  • The temperature
  • The humidity
    • Those data combined can help scientists to predict which disease is going to expand in which area.
    This information is essential to guess which vaccine to deploy where and when.
