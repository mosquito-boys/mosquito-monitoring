The project
What is the purpose of this project ?
This project stands for a tool to help scientists fight monitor mosquitoes propagation.
To achieve this goal, this project is retrieving various data:
  • Mosquito species guessed from their pictures, using Machine Learning
  • Geolocation, asked from users and used to retrieve other data as:
  • The mosquito population density
  • The temperature
  • The humidity

    • Those data combined can help scientists to predict which disease is going to expand in which area.
    This information may be essential in the fight against mosquitoes carried diseases. Knowing which vaccine to deploy where and when would be an amazing advantage!
